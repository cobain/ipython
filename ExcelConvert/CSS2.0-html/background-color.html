background-color

版本：CSS1

语法：

background-color : transparent | color

取值：

transparent : 背景色透明
color : 指定颜色。

说明：

设置或检索对象的背景颜色。
对应的脚本特性为backgroundColor。

示例：

p { background-color: silver }
div { background-color: rgb(223,71,177) }
body { background-color: #98AB6F }
pre { background-color: transparent; }
