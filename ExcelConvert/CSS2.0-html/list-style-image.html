list-style-image

版本：CSS1　

语法：

list-style-image : none | url (url)

取值：

none : 不指定图像
url : 使用绝对或相对地址指定背景图像

说明：

设置或检索作为对象的列表项标记的图像。
若list-style-image属性为none或指定图像不可用时，list-style-type属性将发生作用。
对应的脚本特性为listStyleImage。

示例：

ul.out { list-style-position: outside; list-style-image: url("images/ie.gif"); }
