speech-rate

版本：CSS2.0　

语法：

speech-rate : x-slow | slow | medium | fast | x-fast | slower | faster | number

取值：

x-slow : 80字/分钟
slow : 120字/分钟
medium : 180-200字/分钟
fast : 300字/分钟
x-fast : 500字/分钟
slower : 根据当前的速度每分钟减少40个字
faster : 根据当前的速度每分钟增加40个字
number : 整数。指定每分钟发音字数

说明：

设置或检索发音速度。
目前IE5.5尚不支持此属性。
对应的脚本特性为speechRate。

示例：

strong { speech-rate: fast }
