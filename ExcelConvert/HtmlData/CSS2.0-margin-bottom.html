margin-bottom

版本：CSS1　

语法：

margin-bottom: auto | length

取值：

auto : 值被设置为相对边的值
length : 由浮点数字和单位标识符组成的长度值 | 或者百分数。百分数是基于父对象的高度。

说明：

检索或设置对象底边的外延边距。
内联对象要使用该属性，必须先设定对象的height或width属性，或者设定position属性为absolute。
在IE4+，margin属性不可用于td和tr对象。
外延边距始终透明。
对应的脚本特性为marginBottom。

示例：

body { margin-bottom: 11.5%; }
