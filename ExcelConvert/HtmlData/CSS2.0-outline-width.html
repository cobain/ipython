outline-width

版本：CSS2　

语法：

outline-width : medium | thin | thick | length

取值：

medium : 默认宽度
thin : 小于默认宽度
thick : 大于默认宽度
length : 由浮点数字和单位标识符组成的长度值。不可为负值。

说明：

设置或检索对象外的线条轮廓的宽度。参见outline属性。
目前IE5.5尚不支持此属性。
对应的脚本特性为outlineWidth。

示例：

img { outline-color: orange; outline-style: solid ; outline-width: medium ; }
