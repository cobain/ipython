#id

概述：

根据给定的ID匹配一个元素。

如果选择器中包含特殊字符，可以用两个斜杠转义。参见示例。

参数：

id

用于搜索的，通过元素的 id 属性中给定的值。

示例：

描述:查找 ID 为"myDiv"的元素。

HTML 代码::
<div id="notMe">
<p>id="notMe"</p>
</div>
<div id="myDiv">
id="myDiv"</div>
jQuery 代码:
$("#myDiv");

结果：

[ <div id="myDiv">id="myDiv"</div> ]

描述:查找含有特殊字符的元素。

HTML 代码::
<span id="foo:bar">
</span>
<span id="foo[bar]">
</span>
<span id="foo.bar">
</span>
jQuery 代码:
#foo\\:bar
#foo\\[bar\\]
#foo\\.bar
