data([key],[value])

概述：

在元素上存放数据,返回jQuery对象。

如果jQuery集合指向多个元素，那将在所有元素上设置对应数据。 这个函数不用建立一个新的expando，就能在一个元素上存放任何格式的数据，而不仅仅是字符串。

参数：

key

存储的数据名。

key,value

key:存储的数据名

value:将要存储的任意数据

obj

一个用于设置数据的键/值对

示例：

描述:在一个div上存取数据。

HTML 代码:
<div></div>
jQuery 代码:
$("div").data("blah");  
// undefined $("div").data("blah", "hello");  
// blah设置为hello $("div").data("blah");  
// hello $("div").data("blah", 86);  
// 设置为86 $("div").data("blah");  
//  86 $("div").removeData("blah");  
//移除blah $("div").data("blah"); 
// undefined

描述:在一个div上存取名/值对数据。

HTML 代码:
<div></div>
jQuery 代码:
$("div").data("test", 
{ first: 16, last: "pizza!" }); 
$("div").data("test").first  
//16; $("div").data("test").last  
//pizza!;
