jQuery.data(element,[key],[value])

概述：

在元素上存放数据,返回jQuery对象。

注意：这是一个底层方法。你应当使用.data()来代替。

参数：

element,key,value

element:要关联数据的DOM对象

key:存储的数据名

value:将要存储的任意数据

element,key

element:要查询数据的DOM对象

key:存储的数据名

element

要查询数据的DOM对象

示例：

描述:在一个div上存取数据。

jQuery 代码:
jQuery.data(document.body, 'foo', 52); 
jQuery.data(document.body, 'bar', 'test');
