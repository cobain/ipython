parent([expr])

概述：

取得一个包含着所有匹配元素的唯一父元素的元素集合。

你可以使用可选的表达式来筛选。

参数：

expr

用于筛选的表达式

示例：

描述:查找每个段落的父元素。

HTML 代码::
<div><p>Hello</p>
<p>Hello</p></div>
jQuery 代码:
$("p").parent()
结果:
[ <div><p>Hello</p>
<p>Hello</p></div>]

描述:查找段落的父元素中每个类名为selected的父元素。

HTML 代码::
<div>
<p>Hello
</p>
</div>
<div class="selected">
<p>Hello Again</p>
</div>
jQuery 代码:
$("p").parent(".selected")
结果:
[ <div class="selected">
<p>Hello Again</p></div> ]
