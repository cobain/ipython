[attribute^=value]

概述：

匹配给定的属性是以某些值开始的元素。

参数：

attribute

属性名

value

属性值。引号在大多数情况下是可选的。但在遇到诸如属性值包含"]"时，用以避免冲突

示例：

描述:查找所有 name 以 'news' 开始的 input 元素

HTML 代码::
<input name="newsletter" />
<input name="milkman" />
<input name="newsboy" />
jQuery 代码:
$("input[name^='news']")

结果：
[<input name="newsletter"/>, 
<input name="newsboy"/>]
