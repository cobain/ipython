position( )

概述：

获取匹配元素相对父元素的偏移。

返回的对象包含两个整型属性：top 和 left。为精确计算结果，请在补白、边框和填充属性上使用像素单位。此方法只对可见元素有效。

示例：

描述:获取第一段的偏移

HTML 代码::
<p>Hello</p><p>2nd Paragraph</p>
jQuery 代码:
var p = $("p:first");
var position = p.position();
$("p:last").html( "left: " + 
position.left + ", top: " + 
position.top );

结果：

<p>Hello</p><p>left: 15, top: 15</p>
